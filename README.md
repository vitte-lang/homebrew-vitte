# Homebrew Vitte Tap

Official Homebrew Tap to install the [Vitte programming language](https://github.com/vitte-lang/vitte).

---

## Installation

First, add the tap:

```sh
brew tap vitte-lang/vitte
```

---

## Then install VITTE
```sh
brew install vitte
```

--- 
## Check the version:

```sh
vitte --version
```
